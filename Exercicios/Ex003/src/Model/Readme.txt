Exercicíos:

Defina uma classe para modelar os funcionários do banco.
Sabendo que todo funcionário possui nome e salário, inclua os getters e setters dos atributos.

Crie uma classe para cada tipo específico de funcionário herdando da classe Funcionário.
Considere apenas três tipos especificos de funcionários:

- Gerentes
- Telefonistas
- Secretarias

- Os gerentes possuem um nome de usuário e uma senha para acessar o sistema do banco.
- As telefonistas possuem um código de estação de trabalho.
- As secretarias possuem um número de ramal.

Crie uma classe Main e teste o funcionamento dos três tipos de funcionários criando um objeto
de cada uma das classes.

Suponha que todos os funcionários recebam uma bonificação de 10% do salário.
Acrescente um método calculaBonificacao() na classe pai para realizar esse calculo.

Altere a classe Main para imprimir a bonificação de cada funcionário, além dos dados que já foram impressos.
Suponha que os gerentes recebam uma bonificação maior que os outros, ou seja 60%.
Reescreva o metodo na classe Gerente.
