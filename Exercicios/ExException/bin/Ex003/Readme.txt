Exercicío:

Considere o problema de uma idade menor ou igual a zero.

a) Crie uma classe Pessoa com os atributos nome e idade, com seus respectivos seletores e modificadores.

b) Crie uma exceção chamada IdadeException que herda de Exception

c) Reescreva o modificar (set) de idade para aceitar apenas idades maiores que zero,
se uma idade menor que zero for digitada uma exceção IdadeException deverá ser lançada.

d) Teste o método para verificar se a exceção está sendo lançada e trate-a.

e) Elabore a lógica de ficar repetindo a pergunta ao usuário enquanto ele não digitar uma idade maior que zero.
